# Student Management System

A console-based Student Management System developed in C++ using Object-Oriented Programming and File Handling.

## Features

- Add Student Record
- Display Student Records
- Search Student
- Update Student Details
- Delete Student Record
- Store data using File Handling

## Technologies Used

- C++
- OOP Concepts
- File Handling
- VS Code
- Git & GitHub
